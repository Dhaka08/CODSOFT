## 📁 Projects

### 🚢 Task 1 - Titanic Survival Prediction
Predicted whether a passenger survived the Titanic disaster based on features like age, gender, and passenger class.

**Steps:**
- Exploratory Data Analysis (EDA)
- Handled missing values (Age, Cabin, Embarked)
- Built Logistic Regression & Random Forest models
- Evaluated using accuracy score, confusion matrix & classification report

**Best Model:** Random Forest — **82.68% Accuracy**

---

### 💳 Task 2 - Credit Card Fraud Detection
Detected fraudulent transactions from a dataset of 284,807 real credit card transactions.

**Steps:**
- Analyzed heavily imbalanced data (only 0.17% fraud cases)
- Applied Undersampling to balance the dataset
- Scaled features using StandardScaler
- Built Logistic Regression & Random Forest models

**Best Model:** Logistic Regression — **93.40% Accuracy**

---

### 🌸 Task 3 - Iris Flower Classification
Classified Iris flowers into 3 species — Setosa, Versicolor, and Virginica based on sepal and petal measurements.

**Steps:**
- Exploratory Data Analysis with Pairplots
- Encoded target variable using LabelEncoder
- Built and compared 3 models — Logistic Regression, Random Forest & KNN

**Best Model:** All Models — **100% Accuracy**
